# 🎓 FCI Classroom Premium

A modern, high-end Educational Management System (LMS) inspired by Google Classroom, built with a focus on **Premium User Experience** and **Elegant Design**.

---

## ✨ Features

### 🚀 Core Functionalities
- **Classroom Management**: Create, join, and manage classrooms with unique invitation codes.
- **Dynamic Stream**: Real-time announcements and discussions within each classroom.
- **Assignment System**: Complete workflow for assignments including file uploads (PDF, Images, Docs) and student submissions.
- **Study Materials**: Share resources and documents with students in an organized manner.
- **Interactive Comments**: Engage in discussions on posts and assignments.
- **Member Directory**: View all teachers and students in a specific classroom.

### 💎 Premium UI/UX
- **Glassmorphism Design**: Beautiful transparent interfaces with backdrop-blur effects.
- **Custom Toaster System**: Professional, stackable notifications for every event (Success, Error, Info).
- **Framer Motion Animations**: Smooth, fluid transitions and micro-interactions.
- **Mobile Responsive**: Fully optimized for phones, tablets, and desktops.
- **Professional Modals**: Custom confirmation dialogs replacing standard browser alerts.

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite, Tailwind CSS.
- **Animations**: Framer Motion.
- **Icons**: Lucide React.
- **HTTP Client**: Axios.
- **State Management**: React Context API (Auth & Notifications).

## 🚀 Getting Started

### 1. Prerequisites
- Node.js (v18+)
- NPM or Yarn
- Laravel Backend (running on port 8080)

### 2. Installation
```bash
# Clone the repository
git clone [your-repo-link]

# Navigate to frontend folder
cd frontend

# Install dependencies
npm install
```

### 3. Environment Setup
Ensure your `src/api/axios.js` is pointing to the correct backend URL:
```javascript
baseURL: 'http://127.0.0.1:8080/api'
```

### 4. Running the App
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.
